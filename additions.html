Enhancements to HTML

Javascript enhances HTML by adding an interactive element to documents.

In addition to its content maturation since inception, HTML has been enhanced by Cascading Style Sheets (CSS) and javascript. CSS allows complex formatting that was limited with just HTML, and javascript adds a dynamic component where users can interact with the document. These additions to HTML have made it into the web experience we know today.

An example of javascript in action is shown to the left. A button tag with a javascript function changes the display of the text inside of the document. Without this added functionality, HTML would only be capable of displaying information as a document. Thanks to enhancements like javascript, content becomes interactive and more engaging for users.
